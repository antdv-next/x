Antdv Next X 提供了以下核心技能，帮助您快速构建 AI 对话应用：

### x-sdk-skills

Antdv Next X SDK 核心技能包，包含 useXChat、XRequest 和自定义 Provider 等完整功能。

| 技能            | 版本  | 说明                                                                              | 标签                    |
| --------------- | ----- | --------------------------------------------------------------------------------- | ----------------------- |
| use-x-chat      | 0.0.1 | 专注讲解如何使用 useXChat Hook，包括自定义 Provider 的集成、消息管理、错误处理等  | `chat` `hook` `sdk`     |
| x-chat-provider | 0.0.1 | 专注于自定义 Chat Provider 的实现，帮助将任意流式接口适配为 Antdv Next X 标准格式 | `chat` `provider` `sdk` |
| x-request       | 0.0.1 | 专注讲解 XRequest 的实际配置和使用，基于官方文档提供准确的配置说明                | `request` `sdk`         |

### x-markdown

Antdv Next X Markdown 渲染技能包，提供富文本渲染和对话增强功能。

| 技能       | 版本  | 说明                                                                                                           | 标签  |
| ---------- | ----- | -------------------------------------------------------------------------------------------------------------- | ----- |
| x-markdown | 0.0.1 | 当任务涉及 @antdv-next/x-markdown 的 Markdown 渲染、流式输出、自定义组件映射、插件、主题或聊天富内容展示时使用 | `sdk` |
